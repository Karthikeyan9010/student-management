This project is a full-stack Student Management System developed using Spring Boot for the backend and HTML, CSS, and JavaScript for the frontend.
It provides RESTful APIs to perform CRUD operations on student data, including adding new students, viewing all records, updating details, and deleting entries. 
The frontend interacts with the backend using Fetch API and dynamically displays data in a table format. 
This project helps in understanding backend development, API integration, and database management using Spring Data JPA."
